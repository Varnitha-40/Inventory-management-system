# Inventory Management System (Frontend)

## Description
This is a frontend-based Inventory Management System built using React.
Currently, the application stores data using browser LocalStorage.

## Features
- Add inventory items
- View inventory list
- Basic login functionality
- Data persistence using LocalStorage

## Tech Stack
- React
- JavaScript
- Vite

## How to Run

1. Install dependencies

npm install

2. Start the development server

npm run dev

The app runs at:
http://localhost:5173
